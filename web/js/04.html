变量 -- 就是用来存储信息的

官方点的定义 -- 变量是数据的“命名存储”。我们可以使用变量来保存商品、访客和其他信息。

变量的申明（或者说定义）

现代化的js代码使用 let 和 const 来什么变量
  • let -- 现代的变量声明方式，变量值可以改变
  • const -- 现代的变量申明方式，一旦定义后变量值无法改变，我们也通常叫做 常量
常量其实只是特殊点的变量，常量是赋值后无法改变的变量，使用const来声明

申明变量-变量的命名规则

  • 01 - 变量名称只能包含：数字、字母、$、_
  • 02 - 变量名称首字母不能是数字
  • 03 - 变量名称区分大小写
  • 04 - 变量名称不能使用JavaScript的保留字
  • 05 - 在编程的世界里“字母”包含：汉语等其他非英文字母
  • 06 - 现代JS必须先声明变量后才能赋值，未声明变量就对变量赋值会报错
  • 07 - 现代JS中，使用const申明的变量，如果试图改变其值就会报错

变量赋值举例

    let number;
    number = 1;

    let p1, p2, p3;
    p1 = 'class_p1';
    p2 = 'class_p2';
    p3 = 'class_p3';

    let language = 'zh-CN';

    const LANG_CN = 'zh-CN';
对于不会变更的变量（用const声明的变量）-- 用大写字母命名
