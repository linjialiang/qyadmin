• 大多数情况下，运算符和函数会自动将赋予他们的值转换为正确的类型。
  • 比如， alert() 会自动将任何值都转换为字符串以进行显示。算术运算符会将值转换为数字。
  • 当然在某些情况下，我们需要将值显式地转换为我们期望的类型。

alert 将数值转为字符串

原始类型转换函数

这里只讲解原始类型的转换， object类型转换这里不会涉及
函数 描述
Stting() 将任意类型的变量，转为字符串
Number() 将任意类型的变，转为number类型
Boolean() 将任意类型的变，转为布尔类型（也叫逻辑类型）

数字型转换遵循以下规则

值 转换后的number值
undefined NaN
null 0
true 1
false 0
string “按原样读取”字符串，两端的空白会被忽略。 空字符串变成0。转换出错则输出 NaN。

四则运算中，string类型优先于number类型

        1 + '2' 输出 "12";
        "1" + 2 输出 "12";
        1 + 'abcd' 输出 "1abcd";
