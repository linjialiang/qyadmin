• 大多数情况下，运算符和函数会自动将赋予他们的值转换为正确的类型。
  • 比如， alert() 会自动将任何值都转换为字符串以进行显示。算术运算符会将值转换为数字。
  • 当然在某些情况下，我们需要将值显式地转换为我们期望的类型。

alert 将数值转为字符串

原始类型转换函数

这里只讲解原始类型的转换， object类型转换这里不会涉及
函数 描述
Stting() 将任意类型的变量，转为字符串
Number() 将任意类型的变，转为number类型
Boolean() 将任意类型的变，转为布尔类型（也叫逻辑类型）

数字型转换遵循以下规则

值 转换后的number值
undefined NaN
null 0
true 1
false 0
string “按原样读取”字符串，两端的空白会被忽略。 空字符串变成0。转换出错则输出 NaN。

四则运算中，”加号“作为连字符优先于作为加法运算符

  • 1. 四则运算 会自动将值转换成number类型。
  • 2. 加号，可作为四则运算的加法运算符，也可作为“连字符”，加号会优先选择连字符。
        1 + 2 输出 3；
        1 + '2' 输出 "12";
        "1" + 2 输出 "12";
        1 + 'abcd' 输出 "1abcd";
        2 - "1" 输出 1；

布尔类型转换说明

  • false -- 仅 0、null、undefined、NaN、空字符（""） 5个值
  • true -- 除了转为false的值以外，其余都是ture
  • Boolean('0') 输出为 true。因为 '0' 是字符串，不是数值
  • Boolean(' ') 输出为 true。以为 ' ' 不是空字符串，包含一个空白符
