循环：while 和 for

循环 是一种重复运行同一代码的方法。

循环最怕出现无限循环，所以循环条件一定要注意

“while” 循环

“do…while” 循环

循环体 至少执行一次

“for” 循环

省略语句段

for 循环的任何语句段都可以被省略。
  1. 省略 begin(开始) 语句段： 
    let i = 0; // 我们已经声明了 i 并对它进行了赋值

for (; i < 3; i++) { // 不再需要 "begin" 语句段
alert( i ); // 0, 1, 2
}
  2. 我们也可以移除 step(步骤) 语句段： 
    let i = 0;

for (; i < 3;) {
  alert( i++ );
}
    该循环与 while (i < 3) 等价。
  3. 实际上我们可以删除所有内容，从而创建一个无限循环： 
    for (;;) {
  // 无限循环
}
    请注意 for 的两个 ; 必须存在，否则会出现语法错误。

跳出循环 break

通常条件为假时，循环会终止。
但我们随时都可以使用 break 指令强制退出。

继续下一次迭代 continue

continue 指令是 break 的“轻量版”。它不会停掉整个循环。而是停止当前这一次迭代，并强制启动新一轮循环（如果条件允许的话）。

break/continue 标签

有时候我们需要从一次从多层嵌套的循环中跳出来，这就需要用到 【break/continue 标签】例如：

  1. break 指令可以与标签一起使用，可以从多层嵌套的循环中跳出来。
  2. continue 指令也可以与标签一起使用。在这种情况下，执行跳转到标记循环的下一次迭代。
    outer: for (let i = 0; i < 3; i++) {

    for (let j = 0; j < 3; j++) {

        let input = prompt(`Value at coords (${i},${j})`, '');

        // 如果是空字符串或被取消，则中断并跳出这两个循环。
        if (!input) break outer; // (*)

        // 用得到的值做些事……
    }
    }
    alert('Done!');
