数据类型

JavaScript 的变量可以保存任何类型的数据，并且变量类型可以随意改变，具有这类性质的编程语言我们称之为“动态类型”的编程语言

在 JavaScript 中有八种基本的数据类型

  • Number 类型 -- 包括整数或浮点数以及3个特殊数值（Infinity、-Infinity、NaN）
  • BigInt 类型 -- 表示任意长度的整数（Number 类型无法代表大于 2^53或小于 -2^53的数值）
  • String 类型 -- JavaScript 中的字符串必须被括在引号里。
  • Boolean 类型 -- 也叫逻辑类型，我们通常会根据英文发音叫做布尔类型，仅包含两个值：true 和 false
  • “null” 值 -- 特殊的 null 值不属于上述任何一种类型。它构成了一个独立的类型，只包含 null 值
  • “undefined” 值 -- 特殊值 undefined 和 null 一样自成类型。undefined 的含义是 未被赋值。
  • object 类型 -- object 类型是一个特殊的类型。
  • symbol 类型 -- 用于创建对象的唯一标识符
typeof 运算符 -- typeof 运算符返回参数的类型。当我们想要分别处理不同类型值的时候，或者想快速进行数据类型检验时，非常有用。
